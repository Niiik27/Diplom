Пользователь (User):
- Содержит основную информацию о пользователе.
- Связан с моделью Контакты (Contacts) через связь один к одному (One-to-One).

Контакты (Contacts):
- Содержит контактную информацию пользователя, такую как телефон, социальные сети и мессенджеры.
- Связан с пользователем через связь один к одному (One-to-One).
- Связан с моделью Социальные сети (UserSocial) через связь один ко многим (One-to-Many).
- Связан с моделью Мессенджеры (Messengers) через связь один ко многим (One-to-Many).

Социальные сети пользователя (UserSocial):
- Содержит информацию о социальных сетях, к которым принадлежит пользователь.
- Связан с моделью Контакты через связь один ко многим (One-to-Many).
- Связан с моделью Список социальных сетей (SocialList) через связь многие ко многим (Many-to-Many).

Мессенджеры (Messengers):
- Содержит информацию о мессенджерах, к которым принадлежит пользователь.
- Связан с моделью Контакты через связь один ко многим (One-to-Many).

Список актуальных мессенджеров (MessengerList):
- Содержит список всех возможных мессенджеров, из которых можно выбирать.
- Не имеет прямой связи с другими моделями, используется для выбора из списка при создании или редактировании мессенджеров пользователя.
